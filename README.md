# e2e-autotest-boilerplate

Framework:
- WebdriverIO (v6) in async mode

Runner:
- Jasmine (v3)

Language:
- TypeScript

Features:
- Page Object Pattern
- Page Elements Pattern
- Allure

Runtime environment:
- Node 16
- npm 6
- Chrome 99

## Project setup
```
npm install
```

### Run tests

Before starting, you need to create `.env`(example: `.env-template`).

```
npm start
```

### Generate report
```
npm run generate-report
```

### Open report
```
npm run open-report
```
